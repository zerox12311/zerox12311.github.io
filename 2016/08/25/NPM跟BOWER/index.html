NPM跟BOWER

2016.08.25 Lux Protoss

前言

最近開始使用Node.js，查了一些opensource很多套件都要用到Bower安裝
心中有些疑惑，一樣都是套件管理怎麼不用Node.js內建的NPM就好了呢?
網路上翻過一輪差異後簡單的在這裡做個總結

簡單比較

NPM BOWER
讀取 Package.json Bower.json
建置目錄 node_modules 可自訂
常用管理 後端套件 前端套件
依賴 Node.js NPM

總結

主要是因為BOWER可以自訂建置目錄，所以在前端一些JS/CSS的使用可以建置在相對應的目錄下面，相對於NPM就比較沒那麼自由
所以在前端使用會用Bower打包套件，後端會用NPM打包
